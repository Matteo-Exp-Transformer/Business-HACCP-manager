🎤 MINI-EPACK-PRO HACCP - PRIMA APP VOCALE AL MONDO
=======================================================

📦 INSTALLAZIONE SUPER SEMPLICE PER WINDOWS
==========================================

🔴 PASSO 1: ESTRAI IL FILE ZIP
------------------------------
1. Fai doppio clic sul file "mini-epack-pro-haccp-completo.zip"
2. Clicca "Estrai tutto"
3. Scegli una cartella (es: Desktop)
4. Clicca "Estrai"

🔴 PASSO 2: VERIFICA PYTHON (FACOLTATIVO)
----------------------------------------
1. Premi Windows + R
2. Scrivi "cmd" e premi Invio
3. Scrivi: python --version
4. Se vedi "Python 3.x.x" → PERFETTO!
5. Se dice "comando non trovato" → vai al passo 3

🔴 PASSO 3: INSTALLA PYTHON (SE SERVE)
-------------------------------------
1. Vai su: https://python.org
2. Clicca "Download Python"
3. Installa con le opzioni predefinite
4. IMPORTANTE: Spunta "Add Python to PATH"

🎯 METODO VELOCE - SENZA PYTHON
===============================

🔴 APERTURA DIRETTA (RACCOMANDATO)
---------------------------------
1. Apri la cartella estratta
2. Entra nella cartella "docs"
3. Fai DOPPIO CLIC su "index.html"
4. Si aprirà nel browser → APP PRONTA! 🚀

🎯 METODO CON SERVER LOCALE
===========================

🔴 SE HAI PYTHON INSTALLATO
---------------------------
1. Apri "cmd" (Windows + R, scrivi cmd)
2. Naviga alla cartella: cd /d C:\percorso\alla\cartella\docs
3. Scrivi: python -m http.server 8080
4. Apri browser su: http://localhost:8080

🎤 COME TESTARE IL CONTROLLO VOCALE
===================================

🔴 NELL'APP PRINCIPALE
---------------------
1. Cerca il PULSANTE MICROFONO BLU in basso a destra
2. Clicca il microfono
3. Clicca "Inizia" nel pannello
4. Dai i permessi al microfono quando richiesto
5. PROVA QUESTI COMANDI:
   • "Temperatura frigorifero 2 gradi"
   • "Pulizia banco lavoro completata"
   • "Note: controllare guarnizione porta"

🔴 PAGINA DI TEST SPECIALE
-------------------------
1. Apri il file "test-voice-control.html"
2. È una pagina dedicata solo al test vocale
3. Più semplice per i primi test!

🎯 RISOLUZIONE PROBLEMI
=======================

❌ "L'app non si apre"
--------------------
• Prova a fare doppio clic su "index.html"
• Usa Chrome o Edge
• Assicurati di aver estratto tutto dal ZIP

❌ "Controllo vocale non funziona"
--------------------------------
• Dai i permessi al microfono
• Usa Chrome (migliore supporto)
• Prova la pagina "test-voice-control.html"

❌ "Python non trovato"
----------------------
• Non è obbligatorio! Usa l'apertura diretta
• Oppure scarica da python.org
